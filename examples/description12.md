<h2 style="color:#3F51B5;">📝 Adding Text</h2>
<p>Text boxes can be added in <b>A-Frame</b> to the 3D space to help label things or direct your users' attention to something specific. As with other objects in A-Frame, it can be animated in position, scale, size, etc.</p>

<p>In this example, we have some floating text that says <b>Hello world!</b>. See if you can change the size and color of this text.</p>

<p>Next, we're going to move the text object with the <b>camera</b> object, so it can follow our user as they move around the 3D world. To do this, cut and paste the text line within the camera rig entity.</p>

<ul>
  <li><b>Text Box:</b> Displays floating text in the 3D scene that can be customized for size, color, and position.</li>
  <li><b>Animation:</b> You can animate the text to move, scale, or change color over time.</li>
  <li><b>Camera Rig:</b> Allows the text to follow the user by moving it within the camera's entity.</li>
</ul>

<p>Feel free to experiment with the <b>text size</b> and <b>color</b> to make it more dynamic!</p>
