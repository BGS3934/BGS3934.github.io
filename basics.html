Simple Box Example

This is a simple A-Frame scene with a red box you can look at and interact with.

Code

<a-scene embedded cursor="rayOrigin: mouse">
  <a-box position="0 1.25 -3" rotation="0 45 0" color="#e30613" shadow
         class="clickable"
         event-set__mouseenter="color: #ff4d4d"
         event-set__mouseleave="color: #e30613"
         event-set__click="rotation: 0 360 0; animation__rotate: property: rotation; to: 0 360 0; dur: 1000; easing: linear">
  </a-box>
  <a-plane rotation="-90 0 0" width="10" height="10" color="#ccc"></a-plane>
  <a-sky color="#ececec"></a-sky>

  <a-entity camera look-controls position="0 1.6 0">
    <a-cursor></a-cursor>
  </a-entity>
</a-scene>

What next?

  • Try editing the box’s color or position in the code above.
